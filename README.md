## Overview
This project extracts SpaceX Falcon 9 and Falcon Heavy launch data from Wikipedia using Python.

## Technologies Used
- Python
- Beautiful Soup (web scraping)
- ScrapeGraphAI (AI-based scraping)

## Files
- `f9Extract.py` – Main scraping program
- Report scripts (f9only, b1only, etc.)
- `ProjectSummary.md` – Markdown summary of the project

## Output
All extracted data is formatted as CSV and printed to standard output.
